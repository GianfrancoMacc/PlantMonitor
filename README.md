# PlantMonitor
Plant monitor in a esp8266  the reads values from  a moisture sensor, DHT11 and display them in a web server with additional neoPixel strip form visual 


### Parts:
-Esp8266 NodeMCU

-DHT11

-Moisture Sensor spede-001121

-NeoPixel strip
